# Linguagem Ubíqua

| Termo | Definição |
|:-------|:----------|
| Cliente | Pessoa física ou jurídica proprietária de um ou mais veículos e que solicita serviços à oficina. |
| Ordem de Serviço | Documento que registra o atendimento de um veículo, incluindo diagnóstico, orçamento, execução e entrega. |
| OS | Abreviação de Ordem de Serviço. |
| Orçamento | Estimativa financeira gerada automaticamente com base nos serviços e peças necessários para execução do reparo. |
| Estoque | Conjunto de peças e insumos disponíveis para utilização nas Ordens de Serviço. |
| Serviço | Procedimento técnico executado pela oficina para manutenção ou reparo de um veículo. |
| Veículo | Automóvel pertencente a um cliente e atendido pela oficina. |
| Usuário | Pessoa autenticada no sistema, podendo possuir os perfis Administrador, Atendente, Mecânico, Estoquista ou Cliente. |
| Atendente | Funcionário responsável pelo atendimento inicial, cadastro de clientes, veículos e abertura de Ordens de Serviço. |
| Mecânico | Profissional responsável pelo diagnóstico e execução dos serviços. |
| Estoquista | Funcionário responsável pela gestão do estoque de peças e insumos. |
| Administrador | Usuário com acesso às funcionalidades administrativas do sistema. |
| Documento | Identificador único do cliente, representado por CPF ou CNPJ. |
| Placa | Identificador único do veículo, aceitando os padrões antigo e Mercosul. |
| Diagnóstico | Processo de análise do veículo realizado pelo mecânico para identificar serviços e peças necessárias. |
| Peça | Componente físico utilizado durante a execução de um serviço. |
| Insumo | Material consumido durante a execução de um serviço. |
| Aprovação | Ato realizado pelo cliente para autorizar ou reprovar a execução do orçamento. |
| Execução | Etapa em que os serviços aprovados são efetivamente realizados. |
| Tempo de Execução | Tempo real gasto para realização de um serviço. |
| Tempo Médio de Execução | Média calculada a partir dos tempos reais registrados nas execuções anteriores de um serviço. |
| Status da OS | Estado atual da Ordem de Serviço durante seu ciclo de vida. |
